# Pilot Zeiterfassung

Dieses Projekt ist eine Demo Web App, die auf dem Smartphone wie ein Terminal in der Tasche wirkt.
Der Pilot ist ohne ERP Anbindung und speichert Zustände lokal im Browser.

## Ablauf für GitHub Pages
- Repository anlegen
- Inhalt dieses Ordners ins Repository kopieren
- In GitHub: Settings, Pages, Branch auswählen, Save
- Danach die Pages URL öffnen

## QR Code
Ein QR Code ist nur ein Link zur Pages URL.
Die URL steht erst fest, nachdem GitHub Pages aktiv ist.
Danach kannst du einen QR Code Generator nutzen und den Code ausdrucken.

## Hinweis zur Demo
- Ohne Backend gibt es keine Live Synchronisation zwischen mehreren Geräten.
- Für eine Präsentation auf einem Gerät ist das gewollt und reicht.
